# Projeto---Flexflix

O site tem como objetivo simular um catálogo de filmes e séries e proporcionando interatividade entre opções através de links (botões) e formulários (contato).


## Descrição das páginas

Todas as páginas seguem uma estrutura padrão no layout:
* Barra do topo (localizada a logotipo e barra de pesquisa);
* Barra lateral à esquerda (localizada as opções de inicio, filmes, séries e contato)
* Conteúdo no restante da página (à direita).


  ## Páginas e seus objetivos

* **Página inicial**: Informações padrão de recomendações de streamings;
* **Página de Filmes**: Dedicada à, somente, os filmes disponíveis;
* **Página de Séries**: Dedicada à, somente, as séries disponíveis;
* **Página de Contato**: Apresentando um pequeno formulário (sem destino) para relatar algo.
